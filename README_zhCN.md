# 伊蕾娜

伊蕾娜（日语：イレイナ）是由白石定规创作、あずーる插画的轻小说《魔女之旅》及其衍生作品的登场角色。

| 本名   | イレイナ（Elaina）                          |

| 发色   | 灰发                                    |
| 瞳色   | 琉璃瞳                                   |
| 身高   | 约 150cm                               |
| 三围   | B:99 W:50 H:85/B:95 W:55 H:85（均为自我申报） |
| 年龄   | 14 岁(故事开始)→16~19 岁(主时间线)→25 岁(星霜之旅)   |
| 生日   | 10 月 17 日                             |
| 星座   | 天秤座                                   |
| 出身地区 | 和平国罗贝塔                                |
